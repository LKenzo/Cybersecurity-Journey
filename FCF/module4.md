## Key Points
Malware
- Malicious Software that disrupts, damage, or gain unaithorized access to computer
- Malware Types:
  - Worm -> can be executed without user and spread to other devices
  - Rootkits -> Collection of malicious software that operates near or within the kernel
  - Keylogger -> Program that records user keystore inputs
  - Potentially Unwanted Program (PUP) -> Downloads unwanted program
  - Spyware -> Tracks computer activity and data
  - Adware -> Hides on device and display ads
  - Dialer -> Uses dialing features to run up phone bills
  - Adversarial AI -> Malicious input designed to confuse neural netowork
  - Ransomware -> Virus that encrypts information on a computer
  - Trojan Horse Vrius -> Virus disguised to look like something
  - Dropper -> A type of trojan horse virus designed to install other malware
  - Rogueware/Scareware -> Misleads user by scaring them
  - Botnet Malware -> Infects a host through command and control server
  - Cryptojacking -> Use User's computing resources illegaly to mine crypto

Computer Virus
- Virus operates if human executes its code.
- Destory or corrupting data
- Traits:
  - Invoked by a user
  - Attached to application
  - Designed to spread to other devices on the network
- Types of Virus:
  - Resident -> infects running app
  - Non-resident -> infects closed app
  - Multipartie -> infects many computers
  - Direct Action -> Accesses computer's main memory and infect all files
  - Browser Hijacker -> Changes the browser settings and functionality
  - Overwrite -> Delete and replace data in files
  - Web Scripting -> Attacks web browser security allows injection of malicious code intro browser.
  - file Infector -> Overwrites files when opened
  - Network -> Cripples networks and spreads to devices connected to it.
  - Boot Sector -> Targets the master boot record and inject into the hard disk partition.

Attack Vector
- Method used to exploit a vulnerability.
- Components:
  - Vulnerability
  - Mechanism
  - Pathway
- Mechanisms:
  - Backdoor -> Unauthorized way of gaining access
  - Logic Bomb -> A code that will execute under specific condition
  - Easter egg -> Hidden feature that could leave network or data exposed
  - Droppers
  - Downloader
  - Shellcode -> Code instructions to take control of a compromised computer
  - Code Injection -> Injects code into an app
 
Attack Surface
- represents total number of entry points that the attack vector creates
